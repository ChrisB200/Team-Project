# Crown and Dial

E-commerce website specialising in wristwatches and accessories created using the laravel framework. The project shows key concepts in web development including MVC architecture, integration with databases and authentication.

## Features

- Light/dark mode toggle
- User authentication - register/login/logout
- Product categories 
- Basket to save products for later
- Checkout page to order products 
- Admin page for managing users and editing products


## Tech Stack

- Backend: Laravel (PHP Framework)  
- Frontend: Blade templates, HTML, CSS, Javascript  
- Collaboration software: Git and Github  
- Integrated Development Environment: Visual Studio Code
